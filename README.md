# JPA 실습
JPA를 처음 접하는 분들을 위해 샘플 소스를 제공.

## 개발환경
- JDK 11 (<https://jdk.java.net/archive>)
- Intellij (<https://www.jetbrains.com/ko-kr/idea/download/#section=windows>)
- H2DB (ver. 2.0.204 <https://www.h2database.com/html/download-archive.html>)
### 사용 Lib
- SpringBoot + JPA
- Lombok
- MapStruct
- H2database
- QueryDSL
- commons-lang3
